📐 The Master Theorem

Cheat code for dividing recurrences:
T(n) = aT(n/b) + f(n)

Compare a vs bd
a < bd ➔ Θ(nd)
a = bd ➔ Θ(nd log n)
a > bd ➔ Θ(nlogb a)

4. Graph Traversal: DFS vs BFS

DFS (Depth First Search) BFS (Breadth First Search)
⬇️

Go Deep! explores path to leaf before backtracking.
⬅️➡️

Go Wide! explores all neighbors before going deeper.
Uses a STACK Uses a QUEUE
Best for: Mazes, Acyclicity Best for: Shortest Path
Efficiency: Both are Θ(V + E)

Appendix: Quick Cheat Sheet

Search & Sort

  • Sequential Search O(n)
  • Binary Search O(log n)
  • Bubble/Exchange Sort O(n2)
  • Naïve String Match O(n*m)

Math Rules

  • Sum 1 to n ≈ Θ(n2)
  • Limit f(n)/g(n) = 0 ➔ f = O(g)
  • Limit f(n)/g(n) = c ➔ f = Θ(g)
  • Limit f(n)/g(n) = ∞ ➔ f = Ω(g)
